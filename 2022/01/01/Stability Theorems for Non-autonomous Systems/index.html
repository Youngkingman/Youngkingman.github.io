Stability Theorems for Non-autonomous Systems

Comparison Functions

Definition of class $\mathcal{K}$ functions

A continuous function $\alpha:[0,a)\to[0,\infty)$ is said to belong to class $\mathcal{K}$ if it is strictly increasing and $\alpha(0)=0$. It is of class $\mathcal{K}_\infty$ if $a=\infty$ and $\alpha(r)\to\infty$ as $r\to\infty$

Definition of class $\mathcal{KL}$ functions

A continuous function $\beta:[0,a)\times[0,\infty)\to[0,\infty)$ is said to belong to class $\mathcal{KL}$ if, for each fixed $t$, the mapping $\beta(r,t)$ belong to class $K$ with respect to $r$ and, for each fixed $r$, the mapping $\beta(r,t)$ is decreasing with respect to $t$ and $\beta(r,t)\to0$ as $t\to\infty$.

examples:

  • $\alpha(r)=\arctan(r)$, class $\mathcal{K}$ but not class $\mathcal{K}_\infty$
  • $\alpha(r)=r^c,c\gt0$, class $\mathcal{K}_\infty$
  • $\alpha(r)=\min(r,r^2)$, class $\mathcal{K}_\infty$
  • $\beta(r,t)=r^2e^{-t}$, class $\mathcal{KL}$
  • $\beta(r,t)=\frac{r}{rt+1}$, $\frac{\partial\beta}{\partial r}\gt0$, class $\mathcal{KL}$.

Lemma(Properties)

Let $\alpha_1,\alpha_2\in\mathcal{K}$ on $[0,a),\alpha_3,\alpha_4\in\mathcal{K}_\infty$ and $\beta\in\mathcal{KL}$ and $\alpha_i^{-1}$ denotes the inverse of $\alpha_i$:

  1. $\alpha_1^{-1}$ is defined on $[0,\alpha_1(a))$ and belongs to $\mathcal{K}$
  2. $\alpha_3^{-1}$ is defined on $[0,\infty)$ and belongs to class $\mathcal{K}_\infty$
  3. $\alpha_1\circ\alpha_2$ belongs to class $\mathcal{K}$
  4. $\alpha_3\circ\alpha_4$ belongs to class $\mathcal{K}_\infty$
  5. $\sigma(r,t)=\alpha_1(\beta(\alpha_2(r),t))$ belongs to class $\mathcal{KL}$

Lemma

Let $V:D\to\mathbb{R}$ be a continuous positive definite function defined on a domain $D\subset\mathbb{R}^n$ that contains the origin. Let $B_r\subset D$ for some $r\gt0$. Then there exist $\alpha_1,\alpha_2\in\mathcal{K}$ defined on $[0,a]$ such that:
$$
\alpha_1(\Vert x\Vert)\le V(x)\le \alpha_2(\Vert x\Vert),\forall x\in B_r
\notag
$$
If $D=\mathbb{R}^n$, $\alpha_1$ and $\alpha_2$ are defined on $[0,\infty)$. Moreover, if $V(x)$ is radially unbounded, then $\alpha_1,\alpha_2\in\mathcal{K}_\infty$.

example:

Consider a quadratic positive definite function:
$$
V(x)=x^TPx
\notag
$$
We know that:
$$
\lambda_\min(P)\Vert x\Vert_2^2\le x^TPx\le\lambda_\max(P)\Vert x\Vert_2^2
\notag
$$
and so we can define $\alpha_1(r)=\lambda_\min(P)r^2$ and $\alpha_2(r)=\lambda_\max(P)r^2$.

Non-autonomous Systems

$$
\dot{x}=f(t,x)\notag
$$

$f(t,x)$ is piecewise continuous in $t$ and locally Lipschitz in $x$ for all $t\ge0$ and all $x\in D,(0\in D)$. The origin is an equilibrium point at $t=0$ if
$$
f(t,0)=0,\forall t\ge0
\notag
$$

Definition:

The equilibrium point $x=0$ of the non-autonomous system is:

  • Stable: If $\forall\epsilon\gt0,\exist\delta=\delta(\epsilon,t_0)\gt0$ such that $\Vert x(t_0)\Vert\lt\delta\implies\Vert x(t)\Vert\lt\epsilon$ stands $\forall t\ge t_0\ge0$

  • Uniformly Stable: stable and $\delta$ is independent of $t_0$

  • Asymptotically Stable: stable and $\exist c= c(t_0)$ such that $x(t)\to0$ as $t\to\infty$ , $\forall\Vert x(t_0)\Vert\lt c$

  • Uniformly Asymptotically Stable: if $c$ is independent of $t_0$ and the convergence is uniformly in $t_0$, that is:
    $$
    \forall\eta\gt0\exist T=T(\eta)\gt0:\Vert x(t)\Vert\lt\eta,\forall t\ge t_0+\tau(\eta),\forall\Vert x(t_0)\Vert\lt c
    \notag
    $$

  • Globally Uniformly Asymptotically Stable: if $\delta(\epsilon)$ can be chosen to satisfies $\lim_{\epsilon\to\infty}=\infty$ and :
    $$
    \forall(\eta,c)\exist T=T(\eta,c)\gt0:\Vert x(t)\Vert\lt\eta,\forall t\ge t_0+T(\eta,c),\forall\Vert x(t_0)\Vert\lt c
    \notag
    $$

It’s easier to define uniform stability and uniform asymptotic stability using comparison functions:

  • $x=0$ is uniformly stable if there exists a class-$\mathcal{K}$ function $\alpha(\cdot)$ and a constant $c\gt0$ such that
    $$
    \vert x(t)\vert\le\alpha(\vert x(t_0)\vert)\notag
    $$
    for all $t\ge t_0$ and for every initial condition such that $\vert x(t_0)\vert\le c$

  • uniformly asymptotically stable: if there exists a class $\mathcal{KL}\beta(\cdot,\cdot)$ such that:
    $$
    \vert x(t)\vert\le\beta(\vert x(t_0)\vert,t-t_0)\notag
    $$
    for all $t\ge t_0$ and for every initial condition such that $\vert x(t_0)\vert\le c$

  • globally uniformly asymptotically table: if $c=\infty$

  • uniformly exponentially stable: if $\beta(r,s)=kre^{-\lambda s}$ for some $k,\lambda\gt0$:
    $$
    \vert x(t)\vert\le k\vert x(t_0)\vert e^{-\lambda(t-t_0)}\notag
    $$
    for all $t\ge t_0$ and for every initial condition such that $\vert x(t_0)\vert\le c$.

example:

Consider a nonlinear system:
$$
\dot{x}=(6t\sin t-2t)x
\notag
$$
Its solution is:
$$
\begin{aligned}
x(t) &= x(t_0)exp\left{\int_{t_0}^t(6\tau\sin\tau-2\tau)d\tau \right}\
&=x(t_0)exp\left{6\sin t-6t\cos t-t^2-6\sin t_0+6t_0\cos t_0-t_0^2 \right}
\end{aligned}
\notag
$$
Note $b\sin t-6t\cos t-t^2\le6+6t-t^2\le15$

We have:
$$
\Vert x(t)\Vert\le\Vert x(t_0)\Vert\cdot \exp\left{
15-6\sin t_0+6t_0\cos t_0-t_0^2
\right}\triangleq c(t_0)\cdot\Vert x(t_0)\Vert
\notag
$$
For any $\epsilon\gt0$, choose $\delta(\epsilon,t_0)=\frac{\epsilon}{c(t_0)}$. Then we obtain:
$$
\Vert x(t)\Vert\le c(t_0)\cdot\Vert x(t_0)\Vert\le c(t_0)\cdot\frac{\epsilon}{c(t_0)}=\epsilon
\notag
$$
which implies that the origin is stable. We study the sensitivity of the system solutions due to the changes in $t_0$. Let $t_0=2k\pi$. We examine $x(t)$ at $t=t_0+\pi=(2k+1)\pi$.
$$
\begin{aligned}
x(t_0+\pi)=x(t_0)\exp\left{(4k+1)(6-\pi)\pi \right}
\end{aligned}
\notag
$$
Thus we have:
$$
\frac{x(t_0+\pi)}{x(t_0)}=\exp\left{(4k+1)(6-\pi)\pi \right}\to\infty,as\ k\to\infty
\notag
$$
Therefore, given any $\epsilon\gt0$, there is no $\delta(\epsilon)$ independent of $t_0$ that could satisfy the uniform stability definition.

example:

consider the following system, defined for $t\gt-1$:
$$
\dot{x}=\frac{-x}{1+t}
\notag
$$
Its solution is $x(t)=\frac{1+t_0}{1+t}x(t_0)$. Because $\vert x(t)\vert\le\vert x(t_0)\vert\implies$ the origin is uniformly stable with $\alpha(r)=r$.

The origin is also asymptotically stable, but not uniformly asymptotically stable, because the convergence rate depends on $t_0$.

Time-Dependent Positive Definite Functions

Definition:

  • $V(t,x)$ is said to be positive semidefinite if $V(t,x)\ge0,V(t,0)=0$
  • $V(t,x)$ is said to be positive definite if $V(t,x)\ge W_1(x)$, for some positive definite function $W_1(x)$.
  • $V(t,x)$ is said to be decrescent if $V(t,x)\le W_2(x)$, for some positive definite function $W_2(x)$.
  • $V(t,x)$ is radially unbounded if $V(t,x)\to\infty$ as $\vert x\vert\to\infty$

examples:

Let $x=\begin{bmatrix}x_1 \ x_2\end{bmatrix}$

  • $V(t,x)=(1+t^2)(x_1^2+x_2^2)$. Find $W_1(x)=x_1^2+x_2^2$
  • $V(t,x)=(1+\sin^2t)(x_1^2+x_2^2)$. Find $W_1(x)=x_1^2+x_2^2$ and $W_2(x)=2(x_1^2+x_2^2)$
  • $V(t,x)=\frac{1+t^2}{2+t^2}(x_1^2+x_2^2)$. Find $W_1(x)=\frac{1}{2}(x_1^2+x_2^2)$ and $W_2(x)=x_1^2+x_2^2$

Theorem

Lyapunov Stability for Non-autonomous System

Let the origin $x=0$ be an equilibrium point of $\dot{x}=f(t,x)$ and $D\subset R^n$ be a domain containing $x=0$. Suppose $f(t,x)$ is piecewise continuous in $t$ and locally Lipschitz in $x$ for all $t\ge0$ and $x\in D$. Let $V(t,x)$ be a continuously differentiable function such that:
$$
\begin{aligned}
W_1(x)\le V(t,x)\le W_2(x)\
\dot{V}=\frac{\partial V}{\partial t}+\frac{\partial V}{\partial x}f(t,x)\le0
\end{aligned}
\notag
$$
for all $t\ge0$ and $x\in D$, where $W_1(x)$ and $W_2(x)$ are continuous positive definite functions on $D$. Then, the origin is uniformly stable.

Further theorem

Suppose the assumptions of the previous theorem are satisfied with:
$$
\dot{V}=\frac{\partial V}{\partial t}+\frac{\partial V}{\partial x}f(t,x)\le-W_3(x)
\notag
$$
for all $t\ge0$ and $x\in D$, where $W_3(x)$ is a continuous positive definite function on $D$. Then, the origin is uniformly asymptotically stable. Moreover, if $r$ and $c$ are chosen such that $B_r={\Vert x\Vert\le r}\subset D$ and $c\lt\min_{\Vert x\Vert=r}W_1(x)$, then every trajectory starting in ${W_2(x)\le c}$ satisfies
$$
\Vert x(t)\Vert\le\beta(\Vert x(t_0)\Vert,t-t_0),\forall t\ge t_0\ge0
\notag
$$
for some class $\mathcal{KL}$ function $\beta$. Finally, if $D=R^n$ and $W_1(x)$ is radially unbounded, then the origin is globally uniformly asymptotically stable.

example:

Consider the following nonlinear system:
$$
\begin{aligned}
\dot{x}_1 &= -x_1-e^{-2t}x_2 \
\dot{x}_2 &= x_1-x_2
\end{aligned}
\notag
$$
We consider the following Lyapunov function candidate(“你们别问我这是怎么来的，反正就这样”——梅老师原话)：
$$
V(t,x)=x_1^2+(1+e^{-2t})x_2^2
\notag
$$
Since:
$$
\Vert x\Vert^2=W_1(x)=x_1^2+x_2^2\le V(t,x)\le x_1^2+2x_2^2=W_2(x)\le2\Vert x\Vert^2
\notag
$$
Hence, $V(t,x)$ is positive definite, decrescent, and radially unbounded.

The derivative of $V(t,x)$ along the system trajectories is given by:
$$
\begin{aligned}
\dot{V}(t,x) &= \frac{\partial V}{\partial t}+\frac{\partial V}{\partial x}\cdot f(t,x) \
&= -2e^{-2t}+2x_1\dot{x}_1+2(1+e^{-2t})\dot{x}_2x_2 \
&= -2x_1^2+2x_1x_2-2x_2^2-2e^{-2t}(x_2^2+x_1x_2-x_2x_1+x_2^2) \
&= -2x_1^2+2x_1x_2-2x_2^2-4e^{-2t}x^2_2 \
&\le-(x_1^2+x_2^2) \triangleq-W_3(x)=-\Vert x\Vert^2
\end{aligned}
\notag
$$
Hence the origin is globally uniformly asymptotically stable (exponentially stable)

Linear Time-Varying Systems

$$
\label{LTV}
\dot{x}(t) = A(t)x(t)
$$

the solution can be represented as: $x(t)=\Phi(t,t_0)x(t_0)$

Theorem

The equilibrium point $x=0$ of the linear system $\dot{x}(t)=A(t)x(t)$ is globally uniformly asymptotically stable (GUAS) if and only if:
$$
\Vert \Phi(t,t_0)\Vert\le ke^{-\lambda(t-t_0)},\forall t\ge t_0\ge0
\notag
$$
for some positive constants $k$ and $\lambda$.

Remark that:

  1. for linear systems GUAS$\Leftrightarrow$Exponential stability
  2. for linear time-varying system, GUAS cannot be characterized by the location of the eigenvalues of $A$.

example:

$$
A(t)=\begin{bmatrix}-1+1.5\cos^2t & 1-1.5\sin t\cos t \
1-1.5\sin t\cos t & -1+1.5\sin^2t\end{bmatrix}
\notag
$$

For each $t$, $\lambda_i(A(t))=-0.25\pm 0.25\sqrt{7}j$. Yet the origin is unstable. $\Phi(t,0)$ is given by:
$$
\Phi(t,0)=\begin{bmatrix}e^{0.5}\cos t & e^{-t}\sin t \
-e^{0.5t}\sin t & e^{-t}\cos t \end{bmatrix}
\notag
$$

Corollary:

The system $(\ref{LTV})$ is uniformly asymptotically stable if there exists $\lambda\gt0$ such that:
$$
\lambda_i(A(t)+A^T(t))\le-\lambda,\forall t\ge0,i=1,…,n.
\notag
$$
Proof: Let $V(t,x) = x^Tx$, then:
$$
\begin{aligned}
\dot{V}&=\dot{x}^Tx+x^T\dot{x} \
&=(A(t)x)^Tx+x^TA(t)x \
&= x^T(A^T(t)+A(t))x\le-\lambda x^Tx=-V
\end{aligned}
\notag
$$

Corollary:

Assume that there exists $P(t)\in\mathbb{R}^{n\times n}$ which is continuously differentiable and symmetric, and there exist $0\lt c_1\lt c_2\lt\infty$ such that:
$$
c_1I\le P(t)\le c_2I,\forall t\ge0
\notag
$$
Further assume for some $Q(t)\in\mathbb{R}^{n\times n}$, continuous and symmetric such that:
$$
Q(t)\ge c_3I\gt0,\forall t\ge0 \
\dot{P}(t)+P(t)A(t)+A(t)^TP(t)=-Q(t)
\notag
$$
Then the system $(\ref{LTV})$ is globally uniformly asymptotically stable.

Proof: Let $V(t,x)=x^TP(t)x\implies c_1x^Tx\le V(t,x)\le c_2x^Tx\implies V(t,x)$ is positive definite and decrescent. And :
$$
\begin{aligned}
\dot{V}&=\dot{x}^TP(t)x+x^TP(t)\dot{x}+x^T\dot{P}(t)x \
&=-x^TQ(t)x\le -c_3x^Tx
\end{aligned}
\notag
\notag
$$

Corollary:

Assume that at any time $t\ge0$, the eigenvalues of $A(t)$ all have negative real parts. In addition, if $A(t)$ is bounded, and $\int_0^{\infty}A^T(\tau)A(\tau)d\tau\lt\infty$, then $(\ref{LTV})$ is globally uniformly asymptotically stable.

Barbalat’s Lemma(very important)

How about the asymptotically stability when $\dot{V}(t,x)$ is only negative semi-definite? For autonomous system, LaSalle’s invariance theorem can be used. But it is not valid for non-autonomous systems.

Barbalat’s Lemma is a purely mathematical result concerning the asymptotically properties of functions and their derivatives.

an example: can convergence of $f \implies\dot{f}\to0$ ?

$$
f(t)=e^{-t}\sin(e^{2t})\
\dot{f}=-e^{-t}\sin e^{2t}+\cos e^{2t}\cdot2e^{t}
\notag
$$

Definition: Uniform Continuity(一致连续)

A function $f:\mathbb{R}\to\mathbb{R}$ is said to be uniformly continuous if:
$$
\forall\epsilon\gt0,\exist\delta(\epsilon)\gt0,\forall\vert t_2-t_1\vert\le\delta
\implies\vert f(t_2)-f(t_1)\vert\le\epsilon
\notag
$$

Lemma(Barbalat):

If the differentiable function:

  1. $f(t)$ has a finite limit as $t\to\infty$.
  2. $\dot{f}(t)$ is uniformly continuous. ($\ddot{f}$ is bounded)

Then $\lim \limits_{t\to\infty}\dot{f}(t)=0$.

Note: A sufficient condition for a differentiable function to be uniformly continuous is that its derivative be bounded.

Lemma(“Lyapunov-Like”):

If a scalar function $V(t,x)$ satisfies the following conditions:

  • $V(t,x)$ is lower bounded
  • $\dot{V}(t,x)$ is negative semidefinite ($\dot{V}\le0$)
  • $\dot{V}(t,x)$ is uniformly continuous ($\ddot{V}$ is bounded)

then $\lim\limits_{t\to\infty}\dot{V}(t,x)=0$.

example:

In adaptive control, we will often encounter the following non-autonomous system:
$$
\begin{aligned}
\dot{e} &=-e+\phi(t)\tilde{\theta} \
\dot{\tilde{\theta}} &= -\phi(t)e
\end{aligned}
\notag
$$
where $\phi(t)$ is a bounded continuous function, $e$ and $\tilde{\theta}$ are two states of the closed-loop system, representing the tracking error and parameter error.

Consider the quadratic scalar function:
$$
V=\frac{1}{2}e^2+\frac{1}{2}\tilde{\theta}^2\ge0
\notag
$$
Its derivative:
$$
\begin{aligned}
\dot{V}&=e\dot{e}+\tilde{\theta}\dot{\tilde\theta} \
&=-e^2+e\phi(t)\tilde\theta-\tilde{\theta}\phi(t)e \
&=-e^2\le0
\end{aligned}
\notag
$$
is negative semi-definite. Check $\ddot{V}=-2e(-e+\phi(t)\tilde\theta)$.

Note that $\dot{V}\le0$, which implies that $V(t)\le V(0)$. Therefore $e$ and $\tilde{\theta}$ are bounded. With $\phi(t)$ being bounded, we can conclude that $\ddot{V}$ is also bounded. From Barbalat’s Lemma, we can conclude that $\lim\limits_{t\to\infty}\dot{V}(t)=0$, that is, $\lim\limits_{t\to\infty}e(t)=0$.

Theorem(La Salle-Yoshizawa):

Let $x=0$ be an equilibrium point of $\dot{x}=f(t,x)$ and suppose that $f(t,x)$ is locally Lipschitz in $x$ and uniformly in $t$. Let $V(t,x)$ be a continuously differentiable function such that:
$$
\alpha_1(\vert x\vert)\le V(t,x)\le\alpha_2(\vert x\vert) \
\dot{V}=\frac{\partial V}{\partial t}+\frac{\partial V}{\partial x}f(t,x)\le-W(x)\le0
\notag
$$
$\forall t\ge0,x\in\mathbb{R}^n$, where $\alpha_1(\cdot)$ and $\alpha_2(\cdot)$ are class $\mathcal{K}\infty$ functions and $W(x)$ is a continuous function. Then all solutions of $\dot{x}=f(t,x)$ satisfy:
$$
\lim\limits{t\to\infty}W(x(t))=0
\notag
$$
In addition, if $W(x)$ is positive definite, $x=0$ is globally uniformly asymptotically stable.

Proof:

(Revisit if $f$ has a finite limit, $\dot{f}$ uniformly continuous, then $\dot{f}\to0$

so if $\int_0^t W(x(\tau))d\tau$ has a finite limit, $W(x(t))$ is uniformly continuous, then ${W}(x(t))\to0$)

Note, $W(x)\ge0$, so $\int_0^t W(x)$ is monotonous non-decreasing.

From $\dot{V}\le-W(x(t))$ we know $V(t)-V(0)\le-\int_0^t W(x)$ and thus:
$$
\int_0^tW(x(\tau))d\tau\le V(0)-V(t)\le V(0)
\notag
$$
so monotonous non-decreasing function has upper bound, thus it has a finite limit.

Since $\dot{V}\le0$ and $\alpha_1(\Vert x\Vert)\le V(t,x)\le\alpha_2(\Vert x\Vert)\implies V$ is bounded, $\Vert x\Vert\le\alpha_1^{-1}$ is bounded. Assume that $\Vert x\Vert\le R\triangleq\alpha_1^{-1}(V(0))$. Note that a continuous function is uniformly continuous on a closed set. $W(x)$ is uniformly continuous in $x$. For all $t_2\gt t_1\gt0$, we have:
$$
\vert x(t_2)-x(t_1)\vert=\bigg|\int_{t_1}^{t_2}f(\tau,x(\tau))d\tau\bigg|\le L_R\int_{t_1}^{t_2}\Vert x(t)\Vert d\tau\le L_R\cdot R\vert t_2-t_1\vert
\notag
$$
Then $x(t)$ is uniformly continuous in $t$, and thus $W(x(t))$ is also uniformly continuous in $t$. From Barbalat’s Lemma, $\lim\limits_{t\to\infty}W(x(t))=0$.

Theorem($L_p$ norm)

For function of time, the $L_p$ norm is given by:
$$
\label{lpnorm}
\Vert x\Vert_p=\left(\int_0^\infty\Vert x(\tau)\Vert^pd\tau \right)^{\frac{1}{p}}
$$
for $p\in[1,\infty)$, while:
$$
\Vert x\Vert_\infty=\sup\limits_{t\ge0}\Vert x(t)\Vert
\notag
$$
We say that $x\in\mathbb{L}_p$ when $\Vert x\Vert_p\lt\infty$.

Corollary 0.1

If $x\in\mathbb{L}_2\bigcap\mathbb{L}_\infty$ and $\dot{x}\in\mathbb{L}\infty$, then $\lim\limits{t\to\infty}x(t)=0$.

Proof: Define:
$$
f(t)=\int_0^t \Vert x(\tau)\Vert^2 d\tau
\notag
$$
Since $x\in\mathbb{L}_2$, we can get $f(t)$ has a finite limit as $t\to\infty$. Furthermore:
$$
\dot{f}(t)=\Vert x\Vert^2,\ddot{f}(t)=2x^T(t)\dot{x}(t)
\notag
$$
From the fact that $x,\dot{x}\in\mathbb{L}\infty$, we can get $\ddot{f}(t)\in\mathbb{L}\infty$(bounded), which implies that $\dot{f}$ is uniformly continuous. Therefore from Barbalat’s lemma that $\lim\limits{t\to\infty}\dot{f}(t)=0$, that is, $\lim\limits{t\to\infty}x(t)=0$.

Boundedness and Ultimate Boundedness

The concept of stability in the sense of Lyapunov are formulated with respect to an equilibrium point. Often, systems are designed to operate in the presence of disturbances and other uncertainties.

Until now we have used Lyapunov theory to study the behavior of the system about the equilibrium point.

What happens when the system does not have any equilibrium point ?

We will see that Lyapunov analysis can be used to show boundedness of the solution of the state equation.

example:

$$
\dot{x}=-x+\delta\sin t,x(t_0)=a,a\gt\delta\gt0
\notag
$$

Its solution is given by:
$$
x(t)=e^{-(t-t_0)}x(t_0)+\delta\int_{t_0}^{t}e^{-(t-\tau)}\sin\tau d\tau
\notag
$$
The solution satisfies:
$$
\begin{aligned}
\Vert x(t)\Vert&\le e^{-(t-t_0)}\Vert x(t_0)\Vert+\delta\cdot\int_{t_0}^{t}e^{-(t-\tau)}d\tau \
&=e^{-(t-t_0)}a+\delta(1-e^{-(t-t_0)}) \
&=\delta+(a-\delta)e^{-(t-t_0)}\le\delta+a-\delta=a
\end{aligned}
\notag
$$
which shows that the solution is uniformly bounded. Further, for any number $b\in(\delta,a)$, it can be easily seen that:
$$
\Vert x(t)\Vert\le b,\forall t\ge t_0+\ln\frac{a-\delta}{b-\delta}\notag
$$
The bound $b$, which again is independent of $t_0$, gives a better estimate of the solution after a transient period has passed. In this case, the solution is said to be uniformly ultimately bounded and $b$ is called the ultimate bound.

This can be also done via Lyapunov analysis without the explicit solution of th state equation.

Definition

The solution of $\dot{x}=f(t,x)$ are

  • Uniformly Bounded (UB): if there exists a $c\gt0$, independent of $t_0\ge0$ such that:

$$
\forall a\in(0,c)\exist\beta=\beta(a)\gt0:\Vert x(t_0)\Vert\le a\implies \Vert x(t)\Vert\le\beta,\forall t\ge t_0\notag
$$

  • Globally Uniformly Bounded (GUB): if $a$ can be arbitrarily large

  • Uniformly Ultimately Bounded (UUB): with ultimate bound $b$, if there exists $b\gt0,c\gt0$(independent of $t_0$) such that:
    $$
    \forall a\in(0,c)\exist T=T(a,b)\gt0:\Vert x(t_0)\Vert\le a\implies\Vert x(t)\Vert\le b,\forall t\ge t_0+T\notag
    $$

  • Globally Uniformly Ultimately Bounded (GUUB): if $a$ can be arbitrarily large.

Remark: For a Lyapunov function candidate $V$, if we have $\beta,C\gt0$ such that:
$$
\begin{aligned}
\dot{V}&\le-\beta V+C=-\beta(V-\frac{C}{\beta}) \
\implies&e^{\beta t}\dot{V}+e^{\beta t}\beta V\le C\cdot\beta^{\beta t} \
\implies&\frac{d}{dt}(e^{\beta t}V(t))\le C\cdot e^{\beta t} \
\implies&e^{\beta t}V(t)-e^{\beta t_0}V(t_0)\le C\int_{t_0}^{t}e^{\beta\tau}d\tau \
V(t)&\le e^{-\beta(t-t_0)}V(t_0)+\frac{C}{\beta}(1-e^{-\beta(t-t_0)})
\end{aligned}
\notag
$$

Input-to-state Stability

For a linear time-variant system $\dot{x}=Ax=Bu$ with $A$ Hurwitz. The solution is given by:
$$
x(t)=e^{A(t-t_0)}x(t_0)+\int_{t_0}^{t}e^{A(t-\tau)}Bu(\tau)d\tau
\notag
$$
Using the bound:
$$
\Vert e^{A(t-t_0)}\Vert\le ke^{-\lambda(t-t_0)}
\notag
$$
we conclude that:
$$
\begin{aligned}
\Vert x(t)\Vert&\le ke^{-\lambda(t-t_0)}\Vert x(t_0)\Vert+\int_{t_0}^{t}ke^{-\lambda(t-\tau)}\Vert B\Vert\Vert u(\tau)\Vert d\tau \
&\le ke^{-\lambda(t-t_0)}\Vert x(t_0)\Vert+\frac{k}{\lambda}(1-e^{-\lambda(t-t_0)})\Vert B\Vert\sup\limits_{t_0\le\tau\le t}\Vert u(\tau)\Vert \
&\le ke^{-\lambda(t-t_0)}\Vert x(t_0)\Vert+\frac{k}{\lambda}\Vert B\Vert\sup\limits_{t_0\le\tau\le t}\Vert u(\tau)\Vert
\end{aligned}
\notag
$$
And it is GUAS. This doesn’t hold for a general nonlinear system

example

$$
\dot{x}=-3x+(1+2x^2)u
\notag
$$

when $u=0$, the equilibrium point $x=0$ is GAS. Yet, when $x(0)=2$ and $u(t)=1$, the solution is given by:
$$
x(t)=\frac{3-e^t}{3-2e^t}
\notag
$$
which is unbounded. It even has a finite escape time.

Definition of Input to State Stability(ISS)

The system is $\dot{x}=f(t,x,u)$ is said to be input-to-state stable if there exist $\beta\in\mathcal{KL}$ and $\gamma\in\mathcal{K}$ such that for any initial state $x(t_0)$ and any bounded input $u(t)$, the solution $x(t)$ exists for all $t\ge t_0$ and satisties:
$$
\Vert x(t)\Vert\le\beta(\Vert x(t_0)\Vert,t-t_0)+\gamma(\sup\limits_{t_0\le\tau\le t}\Vert u(\tau)\Vert)
\notag
$$

Remarks:

  1. For any bounded input $u(t)$, the state $x(t)$ will be bounded
  2. As $t$ increases, the state $x(t)$ will be ultimately bounded by a class $\mathcal{K}$ function of $\sup\limits_{t\ge t_0}\Vert u(t)\Vert$
  3. If $u(t)\to0$ as $t\to\infty$ then $x(t)\to0$ as $t\to\infty$
  4. Since for $u(t)=0,\Vert x(t)\Vert\le\beta(\Vert x(t_0)\Vert,t-t_0)\Leftrightarrow x=0$ of the unforced system is GUAS.

Theorem

Let $V:[0,\infty)\times\mathbb{R}^n\to\mathbb{R}$ be a $C^1$ function such that:
$$
\alpha_1(\Vert x\Vert)\le V(t,x)\le\alpha_2(\Vert x\Vert) \
\frac{\partial V}{\partial t}+\frac{\partial V}{\partial x}f(t,x,u)\le-W_3(x),\Vert x\Vert\ge\rho(u)\gt0
\notag
$$
$\forall(t,x,u)\in[0,\infty)\times\mathbb{R}^n\times\mathbb{R}^n,\alpha_1,\alpha_2\in{\mathcal{K}_\infty},\rho(u)\in\mathcal{K}$. $W_3(x)\gt0$ and $C^0$. Then the system $\dot{x}=f(t,x,u)$ is ISS with $\gamma=\alpha_1^{-1}\circ\alpha_2\circ\rho$.

Lemma

Suppose that $f(t,x,u)$ is $C^1$ and globally Lipschitz in $(x,u)$, uniformly in $t$. If the unforced system has a globally exponentially stable equilibrium point at $x=0$, then the system $\dot{x}=f(t,x,u)$ is ISS.
